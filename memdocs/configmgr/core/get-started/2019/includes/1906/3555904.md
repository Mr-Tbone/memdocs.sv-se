## <a name="redesigned-notification-logic-for-newly-available-software"></a>Omdesignad meddelande logik för nyligen tillgänglig program vara

Den **nya program varan som är tillgänglig** visas bara en gång för en användare för ett angivet program och en ny revision. Användaren kommer inte längre att se meddelandet varje gången de loggar in. De ser bara ett annat meddelande för ett program om det har ändrats.
